## Project Overview
Nourishly helps users plan meals efficiently by combining nutrition tracking, recipe management, and grocery list generation into a single app.  
The app focuses on usability, accessibility, and modular architecture to allow for future feature expansion.

**Core Features (Planned):**
- Meal plan creation and editing  
- Automatic grocery list generation  
- Recipe search and ingredient tracking  
- Weekly meal calendar  
- Custom user profiles and preferences

npm ci
